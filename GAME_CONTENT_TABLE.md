## 事件卡总表

| 分类 | 卡牌 | 效果 |
|------|------|------|
| 武器 | 拳头哥 | 目标本局只能用拳头，计分×5补偿 |
| 武器 | 平底锅战神 | 目标本局只能近战武器，计分×3补偿 |
| 武器 | 全队武器互换 | 全队当前武器随机重新分配 |
| 武器 | 重新来过 | 指定一人重新随机当前两把武器 |
| 武器 | 武器升级 | 指定一人随机一把武器升一档 |
| 武器 | 武器降级 | 指定一人随机一把武器降一档 |
| 武器 | 偷梁换柱 | 和指定玩家各选一把武器交换 |
| 计分 | 护盾 | 本局负面运算改为“不变” |
| 计分 | 双倍快乐 | 本局自己最终得分×2 |
| 计分 | 小偷 | 偷取指定玩家本局50%得分 |
| 计分 | 乾坤大挪移 | 自己和指定一人本局分数互换 |
| 干扰 | 夺镜 | 指定一人不能使用4倍以上倍镜 |
| 干扰 | 禁药 | 指定一人不能用饮料和止痛药 |

## 惩罚卡总表

| 惩罚 | 效果 |
|------|------|
| 蹲行者 | 全程只能蹲着移动 |
| 静步大师 | 全程只能静步 |
| 螃蟹步 | 只能侧身移动（只按A/D） |
| 跳跳虎 | 移动时每3秒必须跳一次 |
| 趴地哥 | 前20人后只能趴着打 |
| 开镜行者 | 全程开镜移动 |
| 盲狙 | 不能开镜，只能腰射 |
| 裸眼战士 | 不能捡倍镜，只用机瞄 |
| 裸奔 | 不能穿防弹衣和头盔 |
| 极简主义 | 只能用1级包 |
| 和平主义者 | 前5分钟不能开枪 |
| 独狼 | 不能与队友同乘载具 |
| 步行者 | 前3分钟不能使用载具 |
