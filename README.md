# PUBG 随机武器挑战

四人组队 PUBG 娱乐玩法工具。纯前端，直接打开 `index.html` 即可使用。

## 流程

1. 录入 4 名玩家（支持本地名单快速填入）
2. 选择总局数（3/5局）和抽枪模式（轮流/同时）
3. 每人抽 1 张道具卡
4. 每局开始前选择地图 → 随机 2 人抽惩罚卡
5. 全员抽 2 把武器 + 2 个瞄准镜
6. 按顺序使用道具卡
7. 确认装备后开打
8. 结算伤害 → 排名 → 第 1/4 名各抽 1 张道具卡（非最后一局）

## 计分

- 取伤害个位数 → 随机数学运算（平方、翻倍、归零、10-x、减半 等）→ 结果取个位数 = 得分
- 允许负数
- 两人及以上得分相同：各扣 3 分

## 武器

- 每人 2 把，只能用抽到的枪
- 空投武器（Groza/FAMAS/MK14/AWM/P90/Lynx）概率极低
- 已抽武器自动降权减少撞枪
- 地图专属枪：Bizon(艾伦格) / QBZ95+QBU(萨诺) / G36C(维寒迪) / K2(泰戈) / O12+MP9+Mosin(迪斯顿) / JS9(荣都)

## 瞄准镜

- 每人随机 2 个（2x/3x/4x/6x/8x/15x/混合瞄具）
- 捡不到只能用红点/全息

## 道具卡

| 卡牌 | 效果 |
|------|------|
| 近战挑战 | 目标只能用拳头/近战，计分×4，跳过抽武器 |
| 全队武器互换 | 全队 8 把枪打乱重分 |
| 重新来过 | 指定一人重抽武器+瞄准镜 |
| 偷梁换柱 | 和指定玩家各选一把交换 |
| 护盾 | 负面运算改为不变 |
| 双倍快乐 | 得分×2 |
| 小偷 | 偷取目标 50% 得分 |
| 乾坤大挪移 | 和目标互换分数 |
| 夺镜 | 目标禁用 4 倍以上 |
| 禁药 | 目标禁用饮料和止痛药 |

## 惩罚卡

每局开始随机 2 人抽取，扣 3 分可重抽一次。

蹲行者 / 静步大师 / 螃蟹步 / 开镜行者 / 盲狙 / 裸眼战士 / 裸奔 / 极简主义 / 和平主义者 / 骑行者 / 静音模式

## 技术

纯 HTML + CSS + JS，状态存 localStorage，玩家名单独立保存。
