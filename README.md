# PUBG 随机武器挑战

四人组队PUBG娱乐玩法工具。随机抽武器、抽瞄准镜、事件卡牌、每局惩罚，伤害尾数计分。

纯前端，打开 `index.html` 即可使用。

## 玩法流程

1. 输入4人昵称，选地图，选局数（3局/5局）
2. 每人抽1张事件卡（可保留到后面使用）
3. 每局：使用卡牌(可选) → 抽武器×2 → 抽瞄准镜 → 打PUBG → 输入伤害 → 计分
4. 每局结束后随机一人抽惩罚卡（最后一局除外），第1名和第4名各抽1张事件卡
5. 全部局数结束 → 总分排名

## 计分规则

- 取伤害个位数（如伤害215 → 尾数5）
- 根据主武器类型随机一次数学运算（如 x²、×3、+7、10-x、÷2、归零 等）
- 运算结果取个位数 = 本局得分（允许负数）
- 如果有两人或以上得分相同，这些人各扣3分

## 武器系统

每人抽2把武器（主+副），只能使用抽到的枪。

武器分S/A/B/C四档，档位不在界面展示，仅影响抽取概率：
- **S档**（极难抽到）：Beryl、M416、Groza、FAMAS、MK14、AWM、P90
- **A档**：AKM、AUG、SCAR-L、Mini14、SLR、UMP、Vector、Kar98K、M24 等
- **B档**（常见）：ACE32、M16A4、Mk47、MK12、SKS、MP5K、UZI 等
- **C档**（最容易抽到）：VSS、Tommy Gun、S1897、S686、锯短型霰弹枪

空投武器（Groza、FAMAS、MK14、AWM、P90、Lynx AMR）概率极低。

已被其他玩家抽到的武器概率降低，减少撞枪。

## 瞄准镜

抽完武器后每人随机1个倍镜，游戏中只能用这个倍镜（捡不到就机瞄）。

| 倍镜 | 概率 |
|------|------|
| 红点/全息 | 25% |
| 2倍 | 25% |
| 3倍 | 20% |
| 4倍 | 15% |
| 6倍 | 10% |
| 8倍 | 4% |
| 15倍 | 1% |

## 事件卡

第1局前每人抽1张，之后每局第1名和第4名各抽1张，最多持有2张。

| 卡牌 | 类型 | 效果 |
|------|------|------|
| 近战挑战 👊 | 武器 | 目标只能用拳头/近战，计分×4补偿，跳过抽武器 |
| 全队武器互换 🔄 | 武器 | 全队武器随机重新分配 |
| 重新来过 🎰 | 武器 | 指定一人重新随机两把武器 |
| 偷梁换柱 🔀 | 武器 | 和指定玩家各选一把武器交换 |
| 护盾 🛡️ | 计分 | 本局负面运算改为不变 |
| 双倍快乐 ✖️ | 计分 | 本局最终得分×2 |
| 小偷 🦹 | 计分 | 偷取指定玩家本局50%得分 |
| 乾坤大挪移 🔃 | 计分 | 和指定一人分数互换 |
| 夺镜 🔭 | 干扰 | 指定一人不能用4倍以上倍镜 |
| 禁药 💊 | 干扰 | 指定一人不能用饮料和止痛药 |

## 惩罚卡

每局结束随机选一人抽取（最后一局除外），被选过的人下次概率降低。扣3分可重抽一次。

| 惩罚 | 效果 |
|------|------|
| 蹲行者 🦆 | 全程只能蹲着移动 |
| 静步大师 🤫 | 全程只能静步 |
| 螃蟹步 🦀 | 只能侧身移动 |
| 开镜行者 🔫 | 全程开镜移动 |
| 盲狙 🙈 | 不能开镜，只能腰射 |
| 裸眼战士 👁️ | 不能捡倍镜，只用机瞄 |
| 裸奔 🩲 | 不能穿防弹衣和头盔 |
| 极简主义 🎒 | 只能用1级包 |
| 和平主义者 ☮️ | 前5分钟不能开枪 |
| 骑行者 🚲 | 只能使用自行车作为载具 |

## 技术
