# PUBG 随机武器挑战

四人组队 PUBG 娱乐玩法辅助页。纯前端，无需构建；用浏览器直接打开项目根目录下的 `index.html` 即可使用（部分字体走 Google Fonts，首次打开需能访问外网字体地址）。

## 快速开始

1. 四名玩家各填昵称（可从「玩家名单」一键点选填入）。
2. 在「图池」中勾选本局可能抽到的地图（至少保留 1 张）；每局开始会从图池中**随机**一张，无需每局手动选图。
3. 选择总局数：预设 3 / 5 / 7 局，或「自定义」1～20 局（与设置页输入框上限一致）。
4. 点击「开始游戏」。

## 对局流程

1. **首轮抽卡**：每人翻开 1 张道具卡；每人手牌上限 **2 张**（界面文案有说明）。
2. **本局地图**：从你在图池里勾选的地图中随机一张，并切换页面主题色。
3. **惩罚卡**：随机 **2 名**玩家各抽 1 张惩罚卡，效果本局立即生效；每人可 **扣 3 分重抽 1 次**（可选）。
4. **武器**：全员同时抽取（转盘）；每人 **2 把主武器**，仅可使用本局抽到的枪型；已抽过的枪会在权重上降权，减少全队撞枪；空投级武器（Groza / FAMAS / MK14 / P90 / AWM / Lynx AMR）基础权重很低。
5. **瞄准镜**：每人先随机 **2 枚**候选镜，再在「红点/全息」与这两枚之间 **三选二** 作为本局携带；对局中若捡不到所选镜，按规则仅可用红点/全息（见界面说明）。
6. **本局界面**：按顺序使用道具卡 → 确认装备与限制 →「开始游戏」进入结算页。
7. **结算**：输入每人本局伤害，系统按**伤害个位数**、**枪种对应运算表**与道具效果算分；展示排名后，**非最后一局**时第 1 名与第 4 名各再抽 1 张道具卡。
8. **终局**：全部局数结束后展示总排名与明细；可「再来一轮」清空进度重开。

## 计分要点

- 以伤害**个位数**为底数，再按本局主武器类型（突击步枪 / 精确射手步枪 / 冲锋枪 / 狙击枪 / 霰弹枪）随机一条运算（如不变、加减、翻倍、归零、平方、10−x、减半等）；结果再取**个位数**为当局得分（允许中间量为负）。
- **两人及以上当局得分相同**：每人再扣 3 分（平局惩罚）。
- 具体运算种类与权重见 `js/data.js` 中的 `OPERATIONS`。

## 瞄准镜

随机池包含：2x / 3x / 4x / 6x / 8x / 15x / 多倍率混合瞄具（1x/4x）；每人先抽 2 个再在「红点/全息 + 两枚已抽镜」中三选二。

## 道具卡（名称以界面为准）

| 名称 | 效果摘要 |
|------|----------|
| 拳皇来袭 | 目标本局仅拳头/近战，计分有 ×4 补偿 |
| 全队武器替换 | 全队 8 把枪打乱重分 |
| 重新来过 | 指定一人重抽两把武器与本局瞄准镜流程 |
| 交杯酒 | 与指定玩家各选一把枪交换 |
| 防御 | 本局负面运算改为「不变」 |
| Double | 本局自己最终得分 ×2 |
| 小偷 | 偷取指定玩家本局 50% 得分（向下取整） |
| 交杯茶 | 与指定玩家本局分数互换 |
| 夺镜 | 指定玩家不可用 4 倍及以上倍镜 |
| 禁药 | 指定玩家不可用饮料与止痛药 |

道具卡抽取带「近期不重复」的权重平衡逻辑，详见 `js/cards.js` 与 `DRAW_BALANCE`。

## 惩罚卡

每局 2 人抽取；可扣 3 分重抽一次。当前配置中的名称包括（以游戏内卡片为准）：本质蓝博、勒布朗詹姆斯是也、区、裸奔之人、极简主义、和平使者、骑行者、静音时刻 等，完整列表与描述见 `js/data.js` 中 `PUNISHMENT_CARDS`。

## 技术说明

- **栈**：静态 HTML + CSS + JavaScript，无打包步骤。
- **入口**：`index.html` 依次加载 `js/data.js`、`js/wheel.js`、`js/cards.js`、`js/scoring.js`、`js/app.js`；样式在 `css/style.css`。
- **持久化**：游戏进度写入 `localStorage`（键名含版本后缀，如 `pubg_game_state_v3`）；玩家名单与图池勾选分别独立保存，刷新或关闭页面后可继续上局进度。
